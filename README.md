# WorldNow Blogging Platform

A modern AI-powered blogging platform built with Next.js and Gemini AI. This project allows you to generate, manage, and deploy blog content with ease.

- Built with modern web technologies

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (latest LTS recommended)
- npm or yarn

---

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd <your-project-folder>
2. Install dependencies:
   npm install
3. Create environment file:
   Create a .env file in the root directory and add:
   APP_URL="https://your-domain.com"
   MONGODB_URI=your mongodb URI here.
   PORT=3000
   ADMIN_USERNAME= your ADMIN_USERNAME here
   ADMIN_PASSWORD= your ADMIN_PASSWORD here
   JWT_SECRET= your JWT_SECRET here
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_USER= your SMTP_USER here
   SMTP_PASS= your SMTP_PASS here
   SMTP_SECURE="false"
   CLOUDINARY_CLOUD_NAME= your CLOUDINARY_CLOUD_NAME here
   CLOUDINARY_API_KEY= your CLOUDINARY_API_KEY here
   CLOUDINARY_API_SECRET= your CLOUDINARY_API_SECRET here
